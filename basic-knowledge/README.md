## 项目目录结构

```
│── py_${number}_${message}                 // py的大章节
│   └── __pycache__
│   └── 其他文件                             // 章节相关文件
│   └── xxx.py                              // 章节主要代码
│   └── ${title}:${message}.py              // 章节主要代码,更清晰
│   └── xxx.md                              // 章节总结
│
│── py_module_
│   └── xxx.md                              // py模块总结
│
│── py_module_${number}_${module_name}      // py模块练习及总结
│   └── xxx.py                              // 练习代码
│   └── xxx.md                              // 总结
│
├── README.md
```
